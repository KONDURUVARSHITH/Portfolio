Future Prediction and Analysis of Bitcoin Price

GitHub Repository Report

Project Overview

The project aims to accurately predict Bitcoin prices by analyzing various influencing parameters using linear regression and comprehensive market data analysis. The analysis leverages daily trends within the Bitcoin market over a nine-year period, providing insights into effective price prediction features.

Problem Statement

The primary issue with bitcoin is price volatility. Since there's a rapid growth with cryptocurrency's, the transaction of bitcoin over the network is slower and also very expensive to use it. Therefore, it is important to predict and analyze the bitcoin value.

Project Goal

To elevate the R-Squared value obtained from our comprehensive reference. This improvement is intended to enable more precise predictions and analyses of Bitcoin prices within a specific time frame. The resulting dataset will be a valuable asset for future trading activities, with the ultimate goal of achieving favorable results.

Key Features

  • Linear Regression Model Implementation
    • Price prediction using historical data analysis
    • R-squared value optimization
    • Multiple variable correlation analysis
  • Data Analysis Tools
    • Historical price trend analysis
    • Market volatility assessment
    • Performance metrics evaluation
  • Prediction Accuracy
    • High R-squared value (0.999)
    • Low mean absolute error (194.1)
    • Minimal mean absolute percent error (0.014)

Technical Implementation

  • Data Source: Yahoo Finance (Sept 2014 - Oct 2023)
  • Analysis Tools: Alteryx Designer
  • Statistical Methods: Linear Regression, R-squared Analysis
  • Visualization: Advanced plotting and residual analysis

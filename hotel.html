Hotel Customer Data Visualization Project

Code Repository

Dataset Overview

Source

HotelCustomersDataset.xlsx

Period: 2016-2018

Key Features

  • Customer Demographics
  • Booking Information
  • Revenue Data
  • Stay Duration

1. Revenue Analysis (Line Charts)

Monthly Revenue Trends

  • Created monthly aggregation of Lodging and Other Revenue
  • Implemented dual-line chart with date-based x-axis
  • Added grid lines and removed top/right borders
  • Formatted axes labels and legend

2. Quarterly Analysis (Dual Axis)

Bookings vs Cancellations

  • Aggregated data quarterly using resample("Q")
  • Created dual y-axis plot comparing:
    • Total nights booked (left axis)
    • Total cancellations (right axis)
  • Removed scientific notation
  • Added comprehensive labels and formatting

3. Country Analysis (Bar Charts)

Top 10 Countries Analysis

  • Vertical bar chart for room nights by country
  • Horizontal bar chart for lodging revenue
  • Key Findings:
    • France leads in both metrics
    • Germany and Portugal show strong performance
    • Consistent correlation between nights and revenue

4. Revenue Distribution (Grouped & Stacked)

Top 5 Countries Revenue Analysis

  • Grouped bar chart comparing lodging and other revenue
  • 100% stacked bar chart showing revenue proportions
  • Added 80% reference line for baseline comparison
  • Implemented proper spacing and legends

5. Market Share Analysis (Pie/Donut)

Revenue Distribution Visualization

  • Created pie chart for top 5 countries
  • Combined remaining countries into "Other" category
  • Converted to donut chart with center cutout
  • Added percentage labels and proper formatting

6. Customer Behavior Analysis

Stay Duration vs Revenue Analysis

  • Scatter plot comparing average nights to revenue
  • Bubble chart with size based on nightly revenue
  • Age distribution comparison between Germany and France
  • Implemented proper transparency and overlays

Technical Implementation

Key Libraries Used:

  • pandas for data manipulation
  • matplotlib.pyplot for visualizations
  • datetime for time series handling

Key Insights

  • Clear seasonal patterns in revenue trends
  • Strong correlation between length of stay and revenue
  • Geographic concentration in European markets
  • Distinct age distribution patterns across countries
  • Consistent ratio between lodging and other revenue
